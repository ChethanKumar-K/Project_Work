What is the basic difference between a latch and a flip-flop?

In simple, When a latch gate is changed, it latches or is held in that state until it receives the change input again, even if the input is removed. A flip flop only changes as long as the input remains. When the input is removed (turns off) it automatically reverts to the original state.
