What are opcodes and operands in a microprocessor?

Opcode is an instruction that tells processor what to do with the variable or data written besides it. Operand is a variable that stores data(and data can be a memory address or any data that we want to process). e.g. MVI A,B here instruction MVI is an opcode. A & B are operands.

What is 8085 Microprocessor ?

Intel 8085 Microprocessor is an 8 bit microprocessor. It is a 40 pin I.C. package on a single chip. Its clock speed is about 3MHz. It has 80 basic instructions and 246 opcodes It consists of three main sections an arithmetic and logic unit, a timing and Control unit and set of registers.
